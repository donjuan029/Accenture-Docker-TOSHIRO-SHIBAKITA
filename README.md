# Docker: Utilização Prática no Cenário de Microsserviços

Projeto baseado na live coding **"Docker: Utilização prática no cenário de Microsserviços"**, ministrada por **Denilson Bonatti**, instrutor da **Digital Innovation One (DIO)**.

Este repositório tem como objetivo demonstrar, de forma prática, como utilizar **Docker** em uma arquitetura baseada em **microsserviços**, abordando conceitos fundamentais e exemplos aplicáveis no desenvolvimento moderno de aplicações.

---

## 📚 Sobre o Projeto

Nos últimos anos, o uso de **containers** tem se tornado cada vez mais comum no desenvolvimento e na implantação de aplicações. Ferramentas como o **Docker** permitem criar ambientes isolados, portáveis e consistentes, facilitando o desenvolvimento, testes e deploy de sistemas.

Neste projeto são explorados conceitos importantes, como:

* O papel dos **containers** em arquiteturas de microsserviços
* Como utilizar **Docker** para padronizar ambientes de execução
* Exemplos práticos de aplicação no dia a dia de desenvolvimento
* Implementação de serviços distribuídos utilizando containers

---
